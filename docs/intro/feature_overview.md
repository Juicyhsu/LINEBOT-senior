# 🤖 長輩版 AI 機器人 - 功能總覽與使用指南

這是一份專為長輩設計的智慧型 LINE 機器人，結合了最先進的 AI 技術（Gemini 2.5 Flash, Imagen 3）與實用的生活功能。

## 🌟 核心特色

*   **激勵大師人設**：充滿正能量、會講笑話、有同理心的 AI 陪伴。
*   **連結查證防詐騙**：自動檢測不安全連結，多層次驗證，保護長輩上網安全。
*   **超強圖像生成**：一鍵生成精美圖片，還能製作專屬「長輩圖」。
*   **在地化生活服務**：台灣新聞語音播報、用藥提醒、以及景點行程規劃。

---

## 🛠️ 功能詳細介紹

### 1️⃣ 💬 智能對話與陪伴

*   **功能描述**：具備記憶功能的 AI 聊天夥伴。它不僅會回答問題，還會根據對話內容主動關懷、講笑話逗長輩開心。
*   **技術亮點**：使用 Google Gemini 2.5 Flash 模型，反應快速且具備 7 天對話記憶。
*   **使用情境**：
    *   👴 長輩：「今天心情不太好，覺得很無聊。」
    *   🤖 機器人：「哎呀，怎麼會呢？來聽個笑話開心一下...（講笑話）...加油！心情有沒有好一點？讚喔！」

### 2️⃣ 🖼️ AI 繪圖與長輩圖製作

*   **功能描述**：利用 Google 最新的 Imagen 3 模型生成高品質圖片，或將照片製作成帶有問候語的「長輩圖」。
*   **指令觸發**：
    *   `幫我畫[描述]`：生成圖片（例如：「幫我畫一隻在喝咖啡的貓」）。
    *   `我要做長輩圖`：進入引導模式，上傳照片後選擇文字與樣式。
*   **使用情境**：
    *   👴 長輩想傳早安圖給朋友，但不想用罐頭圖片。
    *   👉 對機器人說「我要做長輩圖」，上傳孫子的照片，加上「早安，平安喜樂」，機器人自動合成精美圖卡。

### 3️⃣ 🛡️ 連結查證與防詐騙

*   **功能描述**：當從其他群組轉傳連結給機器人時，它會自動進行安全檢查（網域信任度、關鍵字掃描），並提供「閱讀重點」或「查證真偽」的選項。
*   **指令觸發**：直接貼上任何網址 (URL)。
*   **使用情境**：
    *   👴 長輩收到標題聳動的連結：「驚爆！吃這個會致癌！」
    *   👉 轉傳給機器人。
    *   🤖 機器人：「⚠️ [警示] 這個網站不在可信名單中。請問您要：1. 查證內容（檢查是否為假新聞） 2. 閱讀摘要（幫您快速抓重點）？」

### 4️⃣ 📰 語音新聞播報

*   **功能描述**：自動抓取台灣各大主流媒體（中央社、聯合報等）的即時新聞，並由 AI 整理成 3 則重點摘要。支援「語音朗讀」功能，讓眼睛看不清楚的長輩用「聽」的。
*   **指令觸發**：`看新聞`、`今日新聞`、`聽新聞`。
*   **使用情境**：
    *   👴 長輩早上起床想知道今天發生什麼大事。
    *   👉 對機器人說「看新聞」。
    *   🤖 機器人列出 3 則頭條摘要，並附上「🔊 語音播報」按鈕。長輩點擊後，機器人用清晰的中文語音朗讀新聞給長輩聽。

### 5️⃣ ⏰ 智慧提醒

*   **功能描述**：設定定時提醒，時間一到會透過 LINE 推播通知。支援語音輸入設定。
*   **指令觸發**：`提醒我[時間][做什麼]`（例如：「提醒我明天早上8點吃血壓藥」）。
*   **使用情境**：
    *   👴 長輩怕忘記瓦斯爐在燒水。
    *   👉 對機器人說：「提醒我 15 分鐘後關火。」
    *   🤖 15分鐘後，手機跳出通知：「⏰ 提醒：關火的時間到囉！」

### 6️⃣ 🗺️ 景點推薦與行程規劃

*   **功能描述**：整合 Google Maps 資料，根據長輩所在位置或指定地點，推薦適合的旅遊景點，並規劃無障礙友善的行程。
*   **指令觸發**：`規劃[地點]行程`（例如：「規劃宜蘭一日遊」）。
*   **使用情境**：
    *   👴 周末子女要帶長輩出去玩，不知道去哪。
    *   👉 長輩問：「規劃宜蘭一日遊」。
    *   🤖 機器人推薦：「清水地熱公園（適合泡腳）、傳統藝術中心（有無障礙坡道）...」，並安排好順路行程。

## 📱 技術架構

*   **核心框架**: Python Flask + LINE Messaging API (SDK v3)
*   **AI 模型**: Google Gemini 2.5 Flash (對話/邏輯), Imagen 3 (繪圖)
*   **雲端服務**: Google Cloud Platform (Speech-to-Text, Text-to-Speech)
*   **資料庫**: SQLite / PostgreSQL (用於儲存提醒與用戶狀態)
*   **部署環境**: Zeabur / AWS
